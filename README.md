# Jogo da Forca em Python

Implementação do clássico Jogo da Forca desenvolvido em Python com interface no terminal.

## Funcionalidades

- Sistema de vidas (5 tentativas)
- Suporte para palavras simples ou compostas
- Tratamento de caracteres especiais (acentos)
- Interface colorida no terminal
- Validação de entradas
- Histórico de tentativas com cores (verde para acertos, vermelho para erros)

## Como usar

1. Execute o programa:
```bash
python jogo_da_forca.py
```

2. Insira a palavra desejada e sua categoria
3. Tente adivinhar a palavra letra por letra
4. Você tem 5 vidas!

## Tecnologias

- Python 3.14.2
- Biblioteca `unidecode` para tratamento de caracteres especiais

## Instalação
```bash
pip install unidecode
```
